03 Agent design
S
Scraper agent
Receives a list of competitor URLs. Uses Playwright in headless mode (block images for speed). Returns cleaned text content + SHA-256 hash. Skips pages where hash matches last run.
fetch_page(url) store_snapshot() hash_content()
D
Diff agent
Loads current + previous snapshot. Runs unified diff on cleaned text sections. Filters noise (nav, footer, cookie banners). Returns only changed semantic sections — pricing, hero, features, CTAs.
load_snapshots() semantic_diff() filter_noise()
A
Analyst agent
Sends diffs to Claude claude-sonnet-4-6 with a structured prompt. Forces JSON output with Pydantic schema: claim, category (pricing/messaging/feature/positioning), confidence, and source_section. No hallucination — every claim must cite its source diff.
call_claude(diff) parse_insights() attach_citations()
Sc
Scorer agent
Fully deterministic — no LLM calls. Scores each insight on novelty (first time seen?), frequency (how many competitors share it?), and recency (days since first observed). Deduplicates similar insights. Returns ranked list.
score_novelty() score_frequency() deduplicate()
R
Recommender agent
Final synthesis step. Reads all scored insights from shared state. Calls Claude with full context to generate: top 3 trends, identified whitespace gaps, and 3 concrete "test this next" recommendations. Returns structured report object.
synthesize(insights) identify_gaps() generate_actions()
O
Orchestrator (LangGraph)
StateGraph that routes: Scraper → Diff → Analyst → Scorer → Recommender. Manages shared AgentState TypedDict. Handles conditional edges (skip diff if no changes). Full execution trace stored in Redis for frontend polling.
StateGraph conditional_edge checkpoint
04 Core code snippets

LangGraph orchestrator setup

python orchestrator.py 
from langgraph.graph import StateGraph
from typing import TypedDict, List

class AgentState(TypedDict):
    competitors: List[str]
    snapshots: dict
    diffs: dict
    insights: List[dict]
    scored: List[dict]
    report: dict

graph = StateGraph(AgentState)

graph.add_node("scraper", scraper_agent)
graph.add_node("diff",    diff_agent)
graph.add_node("analyst", analyst_agent)
graph.add_node("scorer",  scorer_agent)
graph.add_node("recommender", recommender_agent)

graph.set_entry_point("scraper")
graph.add_edge("scraper",  "diff")
graph.add_edge("diff",     "analyst")
graph.add_edge("analyst",  "scorer")
graph.add_edge("scorer",   "recommender")

app = graph.compile()

Analyst agent — Claude structured output

python analyst_agent.py 
import anthropic
from pydantic import BaseModel

class Insight(BaseModel):
    claim: str
    category: str  # pricing/messaging/feature
    confidence: float
    source_section: str

def analyst_agent(state: AgentState):
    client = anthropic.Anthropic()
    insights = []
    for comp, diff in state["diffs"].items():
        resp = client.messages.create(
            model="claude-sonnet-4-20250514",
            max_tokens=1000,
            system=ANALYST_PROMPT,
            messages=[{"role":"user",
                       "content": diff}]
        )
        # parse JSON → Insight objects
        parsed = parse_insights(resp, comp)
        insights.extend(parsed)
    return {"insights": insights}

Scraper agent — Playwright

python scraper_agent.py 
from playwright.async_api import async_playwright
import hashlib

async def fetch_page(url: str) -> dict:
    async with async_playwright() as p:
        browser = await p.chromium.launch()
        page = await browser.new_page()
        # Block images/fonts for speed
        await page.route("**/*.{png,jpg,woff}",
            lambda r: r.abort())
        await page.goto(url, timeout=15000)
        html = await page.content()
        text = extract_semantic(html)
        return {
            "url": url,
            "text": text,
            "hash": hashlib.sha256(
                text.encode()).hexdigest()
        }

08 Top risks + mitigations
HIGH
Scraping blocked during live demo
Fix: Pre-run full analysis 12h before demo. Store results in DB. Demo reads from cache — live scraping is a bonus, not a dependency.
HIGH
Claude returns unstructured / hallucinated insights
Fix: Force JSON schema with response_format. Every claim must include source_section from the diff. If source_section is empty, discard the insight.
MED
LangGraph state gets corrupted mid-run
Fix: Use LangGraph's built-in SQLite checkpointing. Each node writes state atomically. Failed runs restart from last checkpoint.
MED
Team blocked on integration at hour 20
Fix: Define FastAPI response schemas (TypeScript types) in hour 2. Frontend builds against mock JSON that matches the real schema from day 1.
LOW
Redis not available in demo environment
Fix: Celery can run with task_always_eager=True for synchronous fallback. No Redis needed for basic demo.
